# Demo: No Finding But Degraded

This safe synthetic repository is intentionally boring from a normal hygiene
scan perspective. It exists to show that no finding is scoped evidence only.

The paired sample gate decision records a degraded runtime observer limitation,
so the result stays `warn` and `execution_authorized=false` instead of becoming
`allow_limited`.

No real credential, personal information, host path, or raw scanner output is
included.
